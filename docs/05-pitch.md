# Pitch Executivo — Inna AI Copilot Multicanal

## Objetivo do Pitch

Apresentar a Inna como uma solução prática de IA Conversacional aplicada à educação financeira, com integração multicanal, automação low-code, API própria, logs e analytics.

A proposta é demonstrar que a Inna não é apenas um chatbot, mas um MVP funcional de agente inteligente integrado a canais reais.

## Roteiro de Apresentação

### 1. Problema

Muitas pessoas têm dificuldade para entender conceitos básicos de finanças pessoais, como reserva de emergência, CDI, Selic, organização de gastos e metas financeiras.

Além disso, muitos canais de atendimento ainda oferecem respostas genéricas, sem contexto, sem histórico e sem integração entre plataformas.

O desafio é criar uma solução que explique finanças de forma simples, segura e acessível, sem realizar recomendações financeiras indevidas.

---

### 2. Solução

A Inna é uma educadora financeira inteligente multicanal, desenvolvida com IA Generativa, FastAPI, Streamlit e n8n.

Ela responde dúvidas financeiras, utiliza dados fictícios de um cliente demo, registra interações e pode ser acessada por diferentes canais, como Web Chat, Telegram, Gmail e Webhook API.

A Inna não recomenda investimentos. Ela atua como uma assistente educativa, explicando conceitos, ajudando na organização financeira e respeitando regras de segurança.

---

### 3. Demonstração

Durante a demonstração, podem ser feitas perguntas como:

```text
O que é CDI?
```

```text
Onde estou gastando mais?
```

```text
Minha reserva de emergência está boa?
```

```text
Onde devo investir meu dinheiro?
```

A resposta esperada é que a Inna explique os conceitos de forma simples e, quando houver pedido de recomendação, recuse de forma segura e educativa.

## Canais Demonstrados

### Web Chat Streamlit

Interface web para conversar com a Inna, visualizar informações do projeto, histórico, feedbacks e analytics.

### Telegram Bot

Canal conversacional automatizado via n8n, onde o usuário envia uma mensagem e recebe resposta da Inna.

### Gmail Bot

Canal de atendimento por e-mail, onde a Inna recebe uma dúvida e responde automaticamente usando Gmail Trigger, HTTP Request e Reply Message.

### Webhook API

Endpoint para integração com sistemas externos, permitindo que outras aplicações enviem perguntas para a Inna.

## Fechamento

O Projeto 1 entrega um MVP funcional de IA Conversacional Multicanal.

Como evolução, a Inna pode se transformar em uma plataforma cloud com:

* PostgreSQL;
* memória por usuário;
* WhatsApp;
* RAG;
* Power BI;
* n8n 24/7;
* deploy em OCI;
* dashboards executivos.

Essa evolução será o Projeto 2: **Inna Cloud AI Platform**.
